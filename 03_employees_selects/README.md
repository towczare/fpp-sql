03.08  Policz wszystkich pracujących mężczyzn i kobiety urodzonych po 1960.
<details>
<summary>Rozwiązanie...</summary>
<p>

```sql
select count(*)'Pracowncicy urodzeni po 1960', gender
from employees
where year(birth_date) > 1960
group by gender;
```

</p>
</details>
